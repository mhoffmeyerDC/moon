# moonPhase
Calculates lunar properties (enlongation, and ?) and inserts that into dataset from [USNO](http://aa.usno.navy.mil/data/docs/RS_OneYear.php).
Based on algorithm and code from [Roger W. Sinnott, Sky & Telescope, 2006](http://www.skyandtelescope.com/wp-content/observing-tools/moonphase/moon.html).
Code refactored for use in NodeJS [module pattern](https://nodejs.org/api/modules.html#modules_modules).

###Dependencies
* [NodeJS](https://nodejs.org/en/) - Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine.
